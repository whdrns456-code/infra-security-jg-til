- **핵심 개념 1: 디스크 관리 4단계** - Disk 장착 → 파티션 생성(`fdisk`) → 파일시스템 포맷(`mkfs`, `mkswap`) → 마운트(`mount`, `swapon`)
- **핵심 개념 2: MBR 파티션 구조** - 주 파티션(Primary) 4개 제한을 극복하기 위한 확장(Extended) 및 논리(Logical) 파티션의 이해
- **핵심 개념 3: 스왑 메모리(Swap)** - 부족한 RAM 공간을 보조기억장치로 대체하여 시스템 다운(OOM 에러)을 방지하는 비상 공간

## 💻 2. 실습 및 구현 내용
### ✅ 실습 목표
- /dev/nvme0n2 디스크에 1G, 2G, 3G 스왑 파티션을 생성하고 자동 마운트 설정

### 🛠️ 구현 결과
```bash
# 1. 파티션 생성 및 ID 변경 (82: Linux swap)
# fdisk /dev/nvme0n2 실행 후 n(생성), t(타입변경), w(저장)

# 2. 스왑 파일시스템 생성
mkswap /dev/nvme0n2p1
mkswap /dev/nvme0n2p2
mkswap /dev/nvme0n2p3

# 3. 영구 마운트 설정 (/etc/fstab)
# vim /etc/fstab 접속 후 아래 내용 추가
/dev/nvme0n2p2  swap  swap  defaults  0  0
/dev/nvme0n2p3  swap  swap  defaults  0  0

# 4. 설정 즉시 반영 및 확인
swapon -a
swapon -s
free -h

```
