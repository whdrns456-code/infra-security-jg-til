## 📝 1. 오늘의 학습 주제 및 핵심 내용
- **주요 주제:** 네트워크 이론(전송방식, 통신방식, MAC 주소)
- **핵심 개념 1: 전송방식**  
    - __유나캐스트(Unicast)__ : 
    - 1대1 대화 -> 송신자와 수신자가 정해져 있다 목적지 주소(mac/IP)를 하나만 선택해 보낸다. 
    - __브로드캐스트(Broadcast)__ : 
    - 같은 네트워크에 있는 모든 장치에게 데이터를 보낸다(트래픽 부하 발생 가능)
    - __멀티캐스트(Multicast)__ : 
    - 정보를 원하는 특정 그룹에게만 데이터를 보낸다. 
    - __중요**__ : 하나의 LAN일 수 있도 여러 LAN일 수 있다(특정 그룹)

    - __애니캐스트(Anycast)__ : 
    - 내 위치에 가장 가꾼 네트워크 상 경로를 선택한다. 1 : 1 통신
    - 성능 최적화와 최단 경로
    - 일반적인 통신에서는 IP가 각 컴퓨터의 고유한 이름이지만,  애니캐스트에서는 IP가 특정 서비스의 이름이 된다. 
    - 속도 향상과 부하 분산, 장애 복구의 장점을 가진다.  
     
- **핵심 개념 2: 통신방식**
    -  __단방향 통신 (Simplex)__ : 
    - 보내기만하고 한 쪽은 받기만 하는 방식이다 : 라디오, 리모컨 등
    - __반이중 통신 (Half Duplex)__ : 
    - 양방향 통신이 가능하지만 동시에는 안된다. : 예 무전기
    - 동시에 데이터를 보내려고하면 충돌(Collision)이 발생하게 된다. 
    - __전이중 통신 (Full Duplex)__ : 
    - 양방향으로 동시에 데이터를 주고받을 수 있다. 
    - 가장 흔한 인터넷 방식이다 충돌 걱정이 없고 가장 효율적이다
