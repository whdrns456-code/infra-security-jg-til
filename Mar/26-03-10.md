🛠️ 3. 트러블슈팅 (Troubleshooting)
🔍 문제 발생 (Issue)
상황: Wireshark에서 패킷을 캡처했으나 특정 사이트(www.ktword.co.kr)의 트래픽을 찾기 어려움.

에러 메시지: 수많은 패킷 속에서 목적지 IP를 즉시 식별 불가.

💡 원인 분석 (Cause)
도메인 이름(FQDN)만 알고 실제 서버의 IP 주소를 필터에 적용하지 못해 발생한 혼선.

🚀 해결 방법 (Solution)
dns.qry.name == www.ktword.co.kr 필터를 사용하여 DNS 질의 패킷을 먼저 확인.

응답값(A 레코드: 220.118.103.67)을 찾은 뒤 ip.addr == 220.118.103.67로 다시 필터링하여 해결.
